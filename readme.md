# Capi Companion

CLI playground for using substrate chains through [capi](https://github.com/paritytech/capi).

Meant for internal exploration of capi.

## Local Development

```bash
npm install
```
```bash
npm link
```

```bash
npm run dev
```

```bash
caco account new
```
